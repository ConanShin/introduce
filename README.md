# 자기소개 게임

HTML5 Canvas를 활용한 사이드 스크롤 플랫폼 게임 형식의 자기소개 페이지입니다.

## 게임 방법

- **방향키 좌우**: 캐릭터 이동
- **스페이스바**: 점프
- **물음표 박스**: 캐릭터가 박스에 닿으면 자기소개 텍스트가 표시됩니다
- **ESC 또는 점프**: 텍스트 닫기

## 모바일 지원

터치 스크린을 지원합니다:
- 화면 왼쪽 터치: 왼쪽 이동
- 화면 오른쪽 터치: 오른쪽 이동
- 화면 중앙 터치: 점프

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript

## 실행 방법

`index.html` 파일을 웹 브라우저에서 열면 됩니다.
